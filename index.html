Bem-vindo(a) ao NavegaWeb!

Este projeto consiste no desenvolvimento de uma aplicação web completa, que inclui uma Landing Page, telas de Autenticação (Login), Cadastro de Usuário e Gestão Administrativa. A aplicação permite o cadastro de novos usuários, a edição e exclusão de usuários existentes, e a exibição de uma lista de usuários na tela de gestão, usando uma tabela dinâmica.

O NavegaWeb é um protótipo de site que reterá logins de um banco de dados vinculados a grupos de estudos da faculdade. Na parte administrativa, a tela permite ao administrador visualizar, editar ou excluir usuários, garantindo uma gestão completa dos dados. Após o cadastro de um novo usuário, o sistema possibilita que ele faça login e acesse a área de gestão.

O front-end da aplicação foi desenvolvido utilizando HTML, CSS, e Bootstrap para garantir um design responsivo e funcional. Para a comunicação entre o front-end e o back-end, foi utilizado o framework Axios, que faz requisições HTTP para interagir com a base de dados do MongoDB, onde os usuários são armazenados. Já o back-end foi feito utilizando as ferramentas Flask, CORS, dotenv e PyMongo.

Além disso, a aplicação foi desenvolvida em conformidade com as normas da Lei Geral de Proteção de Dados Pessoais (LGPD), garantindo a segurança e privacidade das informações dos usuários.
